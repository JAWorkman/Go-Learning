## Projects

### Password Generator

A CLI tool that generates random passwords with several options:

- Random password (8-16 characters)
- Custom length password
- Custom character set password
- Exclude lowercase, uppercase, digits, or special characters

**Run it:**

```bash
go run Password_Generator.go
```
